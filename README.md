# Genesys Cloud Subscription Viewer

This tool uses the Genesys Cloud API to display billing data from an authorized organization.

## Requirements

Only works with [Authorized Organizations](https://help.mypurecloud.com/articles/about-authorized-organizations/) from an existing organization

## How to use

* Login to your **main** Genesys Cloud organization (only [https://login.mypurecloud.ie](https://login.mypurecloud.ie) right now) in a separate tab
* Clone this repository
* Load the index.html in a browser
